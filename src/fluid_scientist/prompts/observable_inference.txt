Field rules:
- phenomenon: physical phenomenon name, english, snake_case
    examples: vortex_identification, shear_layer_instability,
              turbulent_kinetic_energy, recirculation_zone,
              wall_shear_stress, pressure_fluctuation
- target_quantity: the concrete quantity to compute, english, snake_case
    examples: q_criterion, lambda2, vorticity, tke, reynolds_stress,
              skin_friction_coefficient, cp, velocity_gradient
- temporal_mode: one of {statistical, transient, time_averaged}
    statistical   -> quantities accumulated over time (running averages)
    transient     -> instantaneous fields saved at intervals
    time_averaged -> post-processed mean of saved fields
- statistic: one of {mean, rms, psd, snapshot}
    mean     -> time average
    rms      -> root-mean-square (fluctuation intensity)
    psd      -> power spectral density (frequency content)
    snapshot -> instantaneous field
- function_object: the OpenFOAM function object type that produces this quantity
- fields: list of OpenFOAM field names this quantity depends on
- rationale: one short english sentence explaining why this quantity is relevant

================================================================================
COMMON OPENFOAM FUNCTION OBJECT -> QUANTITY MAPPING
================================================================================
Use this reference table to choose function_object:

Vortex / coherent structures:
  - Q-criterion        -> functionObject: Q          (field: Q)
  - Lambda2            -> functionObject: Lambda2    (field: Lambda2)
  - vorticity          -> functionObject: vorticity  (field: vorticity, magnitude)
  - ensight vortexCore -> functionObject: vortexCore

Velocity / gradients:
  - velocity field     -> functionObject: magU / components (U)
  - velocity gradient  -> functionObject: grad(U)
  - divergence of U    -> functionObject: div(U)

Statistics / turbulence:
  - mean fields        -> functionObject: fieldAverage (UMean, pMean, ...)
  - RMS / fluctuations -> functionObject: fieldAverage (UPrime2Mean, pPrime2Mean)
  - Reynolds stress    -> functionObject: fieldAverage (UPrime2Mean tensor)
  - TKE                -> derived from UPrime2Mean: 0.5 * tr(UPrime2Mean)

Forces / wall quantities:
  - forces / moments   -> functionObject: forces
  - forceCoeffs        -> functionObject: forceCoeffs (Cd, Cl, Cm)
  - wall shear stress  -> functionObject: wallShearStress
  - yPlus              -> functionObject: yPlus
  - wallHeatFlux       -> functionObject: wallHeatFlux

Pressure:
  - pressure field     -> field: p
  - Cp                 -> functionObject: pressureTools / components
  - pressure fluctuation -> fieldAverage pPrime2Mean

Probes / sampling:
  - point probes       -> functionObject: probes
  - line / plane cuts  -> functionObject: surfaces / cuttingPlane
  - sets along line    -> functionObject: sets

Spectral / temporal:
  - PSD                -> use probes + external FFT, or fieldAverage + python post
  - time history       -> functionObject: probes / forces (save per time step)

Flow rate / integral:
  - volumetric flux    -> functionObject: fluxSummary / flowType faceZone
  - mass flow          -> functionObject: fluxSummary

Visualization:
  - streamlines        -> functionObject: streamlines
  - isosurface         -> functionObject: surfaces (iso)
